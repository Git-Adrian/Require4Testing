SQL - Structured Query Language

Was ist SQL?

SQL (Structured Query Language) ist eine standardisierte Sprache, die zur Verwaltung und Manipulation von relationalen Datenbanken verwendet wird. Mit SQL können Sie Daten in einer Datenbank erstellen, lesen, aktualisieren und löschen.

Grundlegende SQL-Befehle

Hier sind einige der grundlegenden SQL-Befehle, die Sie kennen sollten:

  • SELECT: Wird verwendet, um Daten aus einer oder mehreren Tabellen abzurufen.
  • INSERT: Fügt neue Datensätze in eine Tabelle ein.
  • UPDATE: Aktualisiert bestehende Datensätze in einer Tabelle.
  • DELETE: Löscht Datensätze aus einer Tabelle.
  • CREATE TABLE: Erstellt eine neue Tabelle in der Datenbank.
  • DROP TABLE: Löscht eine Tabelle aus der Datenbank.

Erweiterte SQL-Themen

Zusätzlich zu den grundlegenden Befehlen gibt es mehrere erweiterte Themen in SQL, darunter:

  • Joins: Joins werden verwendet, um Daten aus mehreren Tabellen basierend auf einer verwandten Spalte zu kombinieren. Typen von Joins umfassen INNER JOIN, LEFT JOIN, RIGHT JOIN und FULL OUTER JOIN.
  • Unterabfragen: Eine Unterabfrage ist eine Abfrage, die in einer anderen SQL-Abfrage eingebettet ist. Sie kann dazu verwendet werden, komplexere Abfragen zu erstellen.
  • Transaktionen: Transaktionen sind eine Folge von SQL-Befehlen, die als eine Einheit ausgeführt werden. Transaktionen helfen, die Datenintegrität sicherzustellen. Wichtige Befehle sind BEGIN TRANSACTION, COMMIT und ROLLBACK.

Beispieldatenbank

Wir verwenden eine Beispieldatenbank mit den folgenden Tabellen:

  • Kunden (KundenID, Name, Adresse, E-Mail)
  • Bestellungen (BestellID, KundenID, BestellDatum)
  • Produkte (ProduktID, Name, Preis)
  • BestellDetails (BestellID, ProduktID, Menge)

Diese Tabellen sind miteinander verbunden und können verwendet werden, um verschiedene SQL-Abfragen zu üben.
