SQL Datentypen und Tabellen

Anlegen von Tabellen in SQL

Bevor Daten in relationalen Datenbanken gespeichert werden können, muss die dafür vorgesehene Tabellenstruktur angelegt werden. Dieser Schritt entspricht in der objektorientierten Programmierung dem Anlegen von Klassen, bevor konkrete Objekte der Klasse erzeugt werden können. Das Anlegen von Tabellen erfolgt mit Ausdrücken einer Data Definition Language (DDL). Die Sprache SQL sieht dafür eine Menge an Schlüsselwörtern vor, von denen eine Auswahl der im praktischen Einsatz am häufigsten genutzten im Folgenden vorgestellt wird.

Datentypen in SQL

Zu jedem Attribut einer Tabelle in SQL muss ein Datentyp festgelegt werden. Mit der Definition des Datentyps wird automatisch auch der von der Datenbank zu reservierende Speicherplatz festgelegt. Darüber hinaus bestimmt die Auswahl des Datentyps auch, wie die gespeicherten Daten intern vom DBMS organisiert werden.

Texttypen in SQL

Grundsätzlich gibt es Texttypen, die eine fest vorgegebene, eine maximale und eine quasi unbegrenzte Menge an Schriftzeichen speichern.

Art SQL-Datentyp Beispiele
Zeichenketten mit immer genau n Zeichen Länge CHAR(n) Ländercodes: CHAR(2), ISBN-13: CHAR(13)
Zeichenketten mit variabler Länge VARCHAR(n) Bezeichnungen: VARCHAR(55), VARCHAR(160)
Sehr lange Zeichenketten TEXT Texte, Blogeinträge

Zahlentypen in SQL

Für die Speicherung von Zahlen gibt es verschiedene Typen. Für Geldbeträge wird der Datentyp DECIMAL empfohlen.

Art SQL-Datentyp Beispiele
Ganze Zahlen INTEGER Anzahl, Index
Dezimalzahlen DECIMAL(n,m) Geldbeträge: DECIMAL(9,2)
Gleitkommazahlen REAL Messwerte

Datumstypen in SQL

Die folgenden Datentypen werden zur Speicherung von Datums- und Zeitangaben verwendet.

Art SQL-Datentyp Beispiele
Kalenderdaten DATE Datum: 2014-06-14
Zeitangaben TIME Uhrzeit: 13:43:56
Bestimmung eines Zeitpunktes TIMESTAMP 2014-06-14 14:00:02

Weitere Datentypen in SQL

Die folgenden Datentypen werden zur Speicherung von Binärdaten und Boolschen Werten verwendet.

Art SQL-Datentyp Beispiele
Binärdaten VARBINARY(n) Speicherung von Binärdaten (z.B. Bilder)
Boolsche Werte BOOLEAN TRUE/FALSE
