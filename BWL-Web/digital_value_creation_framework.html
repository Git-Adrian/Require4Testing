Digital Value Creation Framework nach Hoffmeister

Einführung in das DVC-Framework

Das Digital Value Creation Framework (DVC-Framework) von Christian Hoffmeister bietet eine umfassende Methode zur Abbildung und Analyse digitaler Geschäftsmodelle, insbesondere plattformbasierter Modelle. Im DVC-Framework spielen technische Infrastruktur und Austauschbeziehungen zwischen Anwendenden eine zentrale Rolle.

Das Framework besteht aus einer Plattform und zehn wesentlichen Elementen:

  • Principals (Kund:innen)
  • Agents (Anbietende von Leistungen)
  • Leistungen
  • Softwareagenten
  • Gratifikationen (Gegenleistungen)
  • Komplementoren (Partner, die Voraussetzungen schaffen)
  • Richtungen (Beziehungen zwischen Akteuren)
  • Transaktionen (Abläufe zwischen Beteiligten)
  • Geschäftsmodellpartner
  • Netzwerke (Zusammenspiel von Beteiligten)

Fallbeispiel: Netflix

Das digitale Geschäftsmodell von Netflix bietet ein praktisches Beispiel für die Anwendung des DVC-Frameworks. Die Analyse konzentriert sich auf drei Geschäftsmodellmuster:

  • Subscription: Monatliche Abonnementgebühr für den Zugang zu Filmen und Serien.
  • Long Tail: Angebot von Nischenfilmen zur Ansprache kleiner Zielgruppen.
  • Flatrate: Unbegrenzter Zugang zu Inhalten für einen festen Preis.

Abbildung 19: Relevante Geschäftsmodellmuster im Netflix-Fallbeispiel

Im Netflix-Fallbeispiel veranschaulichen die Leistungen (Filme und Serien) das Angebot an Kund:innen (Principals). Diese zahlen eine monatliche Gebühr (Gratifikation). Filmstudios (Agents) liefern Netflix Filmdateien und erhalten eine Lizenzgebühr (ebenfalls eine Gratifikation). Die Transaktionen und Richtungen des Frameworks zeigen die Abläufe und Beziehungen zwischen den verschiedenen Beteiligten.

Finanzielle Entwicklung von Netflix

Die finanziellen Ergebnisse von Netflix verdeutlichen die Effektivität des Geschäftsmodells. Im Jahr 2022 erzielte Netflix einen Gewinn von rund 4,49 Milliarden US-Dollar, was einen leichten Rückgang gegenüber dem Vorjahr darstellt. Der Umsatz stieg jedoch auf etwa 31,6 Milliarden US-Dollar, was eine signifikante Steigerung darstellt. Diese Finanzergebnisse zeigen die erfolgreiche Anwendung der Transaktionen und Gratifikationen innerhalb des Geschäftsmodells von Netflix.
